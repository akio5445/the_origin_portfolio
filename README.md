# The origin - 記事投稿サイト

## 説明

* The originはプログラマー向けの記事投稿サイトです。
* 誰でも投稿された記事を観覧することが可能です。
* ログインすることで記事を作成、記事にコメントすることが出来ます。
* 情報発信、日々のアウトプット等にお使いください。

## 推奨されるシステム環境

* Ubuntu 18.04.3
* Ruby 2.5.7
* Rails 5.2.4.1
* sqlite3
